    Grey Space Computing
    Remote
    From ₹25,000 a month

    Job details

    Pay

    • From ₹25,000 a month

    Job type

    • Contractual / Temporary
    • Freelance
    &nbsp;

    Full job description

    • Database Cloning: Perform cloning operations from Oracle Cloud Infrastructure (OCI) to the on-premise HQ environment, if required.
    • Tablespace and Log Management: Regularly monitor and manage tablespaces and database logs to ensure optimal performance and system availability.
    • Error Monitoring: Proactively check and resolve any database errors to maintain system health.
    • Session Monitoring: Identify and resolve locking or blocking sessions to prevent performance degradation.
    • Backup Verification: Monitor backup processes and validate successful completion.
    • Disk Space Monitoring: Ensure sufficient disk space availability to support uninterrupted database operations.
    • Object Validation: Detect and address any invalid database objects.
    • Job Scheduling Oversight: Review and ensure successful execution of scheduled database jobs.
    • Weekly Reporting: Provide status reports every Sunday, confirming the health and performance of the database environment.
    • Availability: Remain available for on-demand support in case of urgent database issues, aligned with the Saudi Arabia time zone.

    Job Type: Freelance
    Contract length: 3 months
